# RT Collectibles
A professional platform for managing and selling a sports card collection.

## Project Goals
- **Inventory Tracking:** Catalog rookie cards, parallels, and graded slabs.
- **Market Integration:** Connect with hobby pricing APIs.
- **Sales Portal:** Facilitate secure transactions for collectors.

## Tech Stack
- **Frontend:** React / Next.js
- **Database:** PostgreSQL
- **Authentication:** Clerk / Auth0
